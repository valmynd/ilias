******************************************************************************
INSTALLATION INSTRUCTIONS   (You need at least ILIAS 4.1.0)
******************************************************************************

1. Copy the the current folder into <ILIAS_directory>/Customizing/global/plugins/Modules/TestQuestionPool/Questions/

	You may need to create that directory, first:
	>>> mkdir -p <ILIAS_directory>/Customizing/global/plugins/Modules/TestQuestionPool/Questions/

2. Access to ILIAS and go to the administration page.

3. Select "Modules, Services and Plugins" in the menu on the right.

4. Hit the "Plugins" tab to administrate the Questions Plugin-Slot

5. When ILIAS update the plugin, hit the "activate" link that will appear instead of the "update" link.

6. Hit "Configure" to access the Editor for creating and administrating units
